Instructions for using the app

The app has two switch modes, one button and two button. You can select which mode using the radio buttons above. The two switch mode uses the 'a' key to move forward and the 'b' key to select. The one switch mode moves forward at regular intervals as defined in the number picker above. In the one switch mode, you press the 'a' key to select.

Both modes use row column scanning to improve input times. The app starts by scanning rows. When select is pressed, it scans through all the columns. If select is pressed while scanning columns, the selected item is clicked. If select is not pressed after all columns have been iterated through, then it returns to scanning rows.

Test description

I conducted 3 tests of 5 equations each. The first test was in 1-switch mode at an interval of 750 ms. The second test was 1-switch mode at an interval of 1500ms. The third test was using 2-switch mode. The following equations were tested. There is an equal distribution of numbers and operations.

  • 58+37=
  • 57/52=
  • 314-209=
  • 1012*306=
  • 48*47+96/86-9=
